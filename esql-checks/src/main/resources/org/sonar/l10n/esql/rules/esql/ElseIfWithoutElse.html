This rule applies whenever an IF statement is followed by one or

more ELSEIF statements, the final ELSEIF shall be followed by an ELSE statement.

The requirement for a final ELSE statement is defensive programming.

Noncompliant Code Example

	IF FALSE THEN
		CALL doSomething();
	ELSEIF FALSE THEN
		CALL doSomething();
	END IF;

Compliant Solution

	IF FALSE THEN
		CALL doSomething();
	ELSEIF FALSE THEN
		CALL doSomething();
	ELSE
		CALL doSomething();
	END IF;
