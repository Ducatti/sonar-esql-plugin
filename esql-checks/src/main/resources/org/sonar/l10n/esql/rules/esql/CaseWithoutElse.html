This rule applies whenever an CASE statement is not followed by an ELSE statement.

The requirement for a ELSE statement is defensive programming.

Noncompliant Code Example

	CASE size
	WHEN 0 THEN
	    SET description = 'small';
	WHEN 1 THEN
	    SET description = 'medium';        
	END CASE;

Compliant Solution

	CASE size
	WHEN 0 THEN
	    SET description = 'small';
	WHEN 1 THEN
	    SET description = 'medium';        
	ELSE
	    SET description = 'unknown';
	    CALL handleError();
	END CASE;
