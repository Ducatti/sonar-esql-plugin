This rule applies to check the parameter count of procedure

The parameter count of the procedure should not be higher than the given threshold value.

Noncompliant Code Example

	CREATE PROCEDURE ProcedureWithTooManyParams(IN Environment REFERENCE,IN EventCode CHARACTER,IN ReplaceValue CHARACTER,
		IN RecipientTo CHARACTER,IN Retries INTEGER,IN SenderFrom CHARACTER,IN ServerHost CHARACTER,IN port INTEGER,
		OUT EventBody CHARACTER, OUT Votes CHARACTER, OUT OutputResult CHARACTER)
	BEGIN

Compliant Solution

	CREATE PROCEDURE Procedure1(IN Environment REFERENCE,IN EventCode CHARACTER,IN ReplaceValue CHARACTER,
		OUT OutputResult CHARACTER)
	BEGIN
