This rule applies to check the parameter count of function

The parameter count of the function should not be higher than the given threshold value.

Noncompliant Code Example

	CREATE FUNCTION TempFunction(IN Environment REFERENCE,IN EventCode CHARACTER,IN ReplaceValue CHARACTER,
		IN RecipientTo CHARACTER,IN Retries INTEGER,IN SenderFrom CHARACTER,IN ServerHost CHARACTER,IN port INTEGER,
		OUT EventBody CHARACTER, OUT Votes CHARACTER, OUT OutputResult CHARACTER) RETURNS BOOLEAN
	BEGIN

Compliant Solution

	CREATE FUNCTION TempFunction(IN Environment REFERENCE,IN EventCode CHARACTER,IN ReplaceValue CHARACTER,
		OUT OutputResult CHARACTER) RETURNS BOOLEAN
	BEGIN
